About PooPoo API

PooPoo API is a very bad Application Programming Interface ("API"). We are working hard to make your coding life easier by providing some pre-made codes that you can use in your Discord Bots, websites, etc. Some crediting would be nice, but not mandatory. Also, this is a non-profit and open-sourced project, so any supports will sure be greatly appreciated!
